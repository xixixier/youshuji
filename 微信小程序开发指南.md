# 有数记微信小程序开发指南

## 📋 项目概述

「有数记」是一个帮助用户管理订阅服务的微信小程序，采用现代化的设计风格，提供直观的订阅管理、数据分析和提醒功能。

## 🎨 设计规范

### 色彩系统
- **主色调**：微信绿 `#07C160` - 用于按钮、选中状态、图标高亮
- **背景色**：浅灰 `#F7F7F7` - 页面背景
- **卡片背景**：白色 `#FFFFFF` - 所有内容卡片
- **文字颜色**：
  - 主文本：`#333333`
  - 次文本：`#666666`
  - 辅助文本：`#999999`
- **分隔线**：`#F0F0F0`
- **边框**：`#E5E5E5`

### 字体规范
- 系统默认字体，优先使用 PingFang SC
- 标题字重：600
- 正文字重：400
- 辅助信息字重：400

### 间距规范
- 页面水平边距：16px
- 卡片内边距：20px
- 元素间距：12px、16px、20px
- 圆角规范：8px、12px

## 🔄 核心功能逻辑

### 订阅日期计算
```javascript
// 推算下次扣费日期
function calculateNextPayment(firstDate, cycle) {
  const now = new Date();
  let nextDate = new Date(firstDate);
  
  while (nextDate < now) {
    switch (cycle) {
      case 'week': 
        nextDate.setDate(nextDate.getDate() + 7); 
        break;
      case 'month': 
        nextDate.setMonth(nextDate.getMonth() + 1); 
        break;
      case 'quarter': 
        nextDate.setMonth(nextDate.getMonth() + 3); 
        break;
      case 'year': 
        nextDate.setFullYear(nextDate.getFullYear() + 1); 
        break;
    }
  }
  
  return nextDate;
}
```

### 支出统计计算
```javascript
// 计算月度总支出
function calculateMonthlyTotal(subscriptions) {
  return subscriptions
    .filter(item => !item.isExpired)
    .reduce((total, item) => total + Number(item.price), 0);
}
```

## 🔧 开发要点

### 微信云数据库操作
```javascript
// 获取数据库引用
const db = wx.cloud.database();

// 查询订阅列表
db.collection('subscriptions')
  .orderBy('createdAt', 'desc')
  .get()
  .then(res => console.log(res.data));

// 添加订阅
db.collection('subscriptions').add({
  data: { /* 订阅数据 */ }
});

// 更新订阅
db.collection('subscriptions').doc(id).update({
  data: { /* 更新数据 */ }
});

// 删除订阅
db.collection('subscriptions').doc(id).remove();
```

### 用户授权
```javascript
// 获取用户信息
wx.getUserProfile({
  desc: '用于完善用户资料',
  success: (res) => {
    console.log(res.userInfo);
  }
});
```

### 消息订阅（提醒功能）

```javascript
// 请求消息订阅权限
wx.requestSubscribeMessage({
  tmplIds: ['模板ID'],
  success: (res) => {
    console.log('订阅成功', res);
  }
});
```

## 🤔 常见问题

### Q: 如何修改主题色？
A: 在 `app.wxss` 中修改 CSS 变量 `--wechat-green` 即可全局生效。

### Q: 如何添加新的分类？
A: 在首页和添加页的分类数组中添加新分类项。

### Q: 数据备份功能如何实现？
A: 可以使用云函数将用户数据导出为 Excel/CSV，提供下载链接。
