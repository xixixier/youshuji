# 有数记｜开发文档（工程约定与路线图）

## 概览
- 目标：以极简黑白灰 + Apple 风格呈现订阅管理（成本、续费、统计）。
- 主要页面：
  - 首页（pages/index）：数据拉取、分类筛选、即将续订、删除/编辑、统计入口。
  - 添加（pages/add）：表单、图标半屏弹窗、联网检索、上传图标、保存至云端。
  - 编辑（pages/edit）：数据回显与更新、订阅消息授权拉起。
  - 统计（pages/stat）：分类聚合、月度折算、百分比进度条动画。
- 云端能力：
  - 云数据库：subscriptions 集合。
  - 云函数：sendReminder 定时检查并下发订阅消息。

## 技术栈与依赖
- 微信小程序（WXML/WXSS/JS），微信云开发（数据库/云函数/openapi）、Haptics。
- 外部服务（可选增强）：Apple iTunes Search API、F‑Droid Search API。
- 不使用第三方图表库（统计页为纯 CSS 动画）。

## 目录结构（节选）
- miniprogram/pages/index：首页逻辑与样式
- miniprogram/pages/add：添加订阅页、半屏弹窗、搜索与上传
- miniprogram/pages/edit：编辑页与授权拉起
- miniprogram/pages/stat：统计页（聚合与可视化）
- cloudfunctions/sendReminder：定时推送云函数
- DESIGN.md：设计规范基线（Apple 风格）
- DEVELOPMENT.md：本开发文档

## 环境与配置
- 微信开发者工具（开启云开发）。
- 云环境 ID 已在 app.js 初始化。
- 合法域名（若启用在线检索）：
  - `https://itunes.apple.com`
  - `https://search.f-droid.org`
- 若真机依然请求受限，功能会回落至本地热门库与别名匹配（不影响添加）。

## 数据模型（subscriptions）
- 基础字段：
  - appName（String）：应用名
  - price（Number）：金额
  - currency（String）：货币符号，如￥
  - purchaseType（'sub' | 'buyout'）
  - cycle（'week' | 'month' | 'quarter' | 'year' | 'custom' | null）
  - firstDate（String：YYYY-MM-DD）
  - endDate（String：YYYY-MM-DD，仅 custom）
  - isRemind（Boolean）：是否提醒
  - category（String）：分类（影音娱乐/实用工具/学习办公/游戏/其他…）
  - remark（String）：备注
  - iconUrl（String）：图标 URL 或云文件 fileID
  - createdAt（ServerDate）
  - isCancelled（Boolean，可选）
- 前端计算派生：
  - daysRemaining、formattedNextPayment、isUrgent（≤3 天）、isExpired（已结束）

## 页面逻辑说明
- 首页（index）
  - 拉取后先清洗图标：icons8 或空链接强制置空并标记 iconError。
  - 总支出：只要有 price 且未过期即计入（包含买断制按总额计入首页总额）。
  - 即将续订：订阅制 + 30 天内 + 按剩余天数升序；紧急（≤3 天）微红提示；过期卡片灰态并沉底。
  - 分类筛选：allSubscriptions 前端缓存，按 currentCategory 过滤展示。
  - 交互：卡片点击 → ActionSheet（编辑/删除）→ 二次确认 → 云端删除 → 刷新。
  - 统计入口：顶部黑卡“查看统计”跳转 /pages/stat/stat。
- 添加页（add）
  - 表单与动态字段：订阅制/买断制、自定义周期日期等。
  - 图标弹窗：
    - 防抖 500ms；本地热门库即时命中；Apple（CN/US）与 F‑Droid 合并去重；有图优先覆盖。
    - 支持上传自定义图标（云存储），选择后自动回填并关闭弹窗。
  - 保存：字段校验 → db.add；成功后拉起 `wx.requestSubscribeMessage` → Toast → 返回。
  - 分类写入：若未选择库项默认“其他”。
- 编辑页（edit）
  - onLoad 拉取 doc(id) 完整回显；handleUpdate 更新数据。
  - 更新成功同样拉起订阅消息授权 → Toast → 返回。
- 统计页（stat）
  - 过滤掉已过期；按分类聚合月度金额（周×4.33，季÷3，年÷12；买断制按 12 个月平摊）。
  - 计算百分比并排序；黑卡展示总额；白卡内纯 CSS 进度条动画（transition: width 1s）。

## 设计规范要点（对齐 DESIGN.md）
- 单一互动色：Apple Blue `#0071e3`，只用于互动元素；主 CTA 保留黑色。
- 标题排版：紧致行高（~1.1）与轻微负字距，正文保持舒适行高。
- 黑白灰大圆角卡片，阴影极少；不使用渐变纹理。
- 触控反馈：轻震动、柔和缩放。

## 云函数与订阅消息
- sendReminder（cloudfunctions/sendReminder）
  - 触发：定时器 `"0 0 10 * * * *"`（每日 10:00）。
  - 策略：遍历订阅制，基于 firstDate + cycle 推算 nextPaymentDate；3 天内发送模板消息。
  - 模板字段需与微信后台一致（thing1/amount2/time3 仅为占位示例）。
  - 部署方式：开发者工具右键云函数 → 上传并安装依赖（含触发器）。
- 客端授权拉起：保存/更新成功后调用 `wx.requestSubscribeMessage`（模板 ID 同步后台）。

## 开发约定
- 不新增额外强调色；仅 Apple Blue 用于可交互强调，主按钮使用黑色。
- 标题使用紧致行高与负字距；按钮高度紧凑、按压有反馈。
- 搜索防抖 500ms；优先本地热门库命中；接口结果合并后有图替换无图。
- 过期订阅前端灰态且沉底；已过期不计入首页总支出与统计月度总额。

## 常见问题与排查
- 搜索真机无结果
  - 检查“服务器域名 → request 合法域名”是否包含 `https://itunes.apple.com` 与 `https://search.f-droid.org`。
  - 若被限制，功能仍回落至本地热门与别名；可考虑补一份离线图标映射增强稳定性。
- scroll-view 不滚动
  - 弹窗容器需 `flex` + 固定高度；内部 `scroll-view` 需 `flex:1;height:0;`。
- 图标 404
  - 任一图标拉取失败即走首字母灰底兜底，不影响交互。
- 模拟器/真机差异
  - 字体度量与网络策略不同，已在样式与搜索策略中做兼容与回退。

## 路线图（建议）
- 落库 nextPaymentDate：在保存/更新时生成并持久化，降低云函数计算复杂度。
- 离线图标映射：内置常见国内外应用图标映射表（SVG/小图），完全脱离外网依赖。
- 设置页：默认货币与提醒时间偏好、开关订阅消息等。
- 事件与错误上报：关键路径埋点与失败日志。

- 首页：
  - 总支出包含未过期的全部价格（包含买断制）；分类筛选正确。

## 本地开发与联调指引
### 开发者工具项目设置
- 绑定云开发环境；开启“上传代码时样式自动补全/压缩”以减小包体。
- 关闭“ES6 转 ES5”除非目标机型需要；避免运行时多余 polyfill。
- 真机调试前，先在“详情 > 本地设置”关闭“不校验合法域名、TLS 版本及 HTTPS 证书（仅调试）”，以模拟真实线上环境。

### 真机调试步骤
1. 在首页下拉刷新，确认云端拉取与分类筛选正常。
2. 进入添加页：
   - 打开半屏图标弹窗，搜索中文/英文别名（例：腾讯视频/qq music/netflix）。
   - 无网络或接口失败时，热门库与别名应即时命中并可选。
   - 选择图标或上传图片，确认回填成功。
3. 保存后观察订阅消息授权弹窗；点击“允许/拒绝”均应 Toast 成功并返回首页。
4. 编辑页回显并修改；保存后同样拉起授权并返回。

### 合法域名与网络自检
- 公众号后台：开发 > 开发管理 > 开发设置 > 服务器域名 > request
  - 必须包含：`https://itunes.apple.com`、`https://search.f-droid.org`
  - 仅域名，无路径、端口、参数；需有效证书。
- 真机若出现 `url not in domain list`，说明未生效；若超时/证书错误，可能是网络或目标站限制。
- 所有联网失败都不影响添加：回退到本地热门库 + 别名匹配 + 首字母兜底。

## 图标与图片规范
- 图标优先来源：Apple（CN/US）与 F‑Droid；若不可达，使用热门库与上传图标。

## 样式与排版细则
- 唯一互动色：`#0071e3`；主 CTA 按钮保持纯黑 `#000000`，按下转 `#1d1d1f` 并轻微缩放。

- 滚动容器（如半屏弹窗）：父容器固定高度 + `display:flex` + 子级 `scroll-view{flex:1;height:0;}`。

## 代码规范与约定（详细）
### JS/WXML/WXSS
- 命名：小写驼峰（变量/方法），大写首字母（构造/类），连字符类名（WXSS）。
- 结构：页面数据统一定义在 `data` 顶部，事件方法下方，工具函数置底。
- 条件渲染：优先使用 `wx:if` 控制大块区域，`hidden` 只用于简单显隐。
- 列表渲染：提供稳定 `wx:key`（如 `_id`），避免使用 `index` 作为 key。
- 样式单位：移动端优先 `rpx`；仅在必须与像素一一对应时使用 `px`。
- 交互反馈：点击态添加轻震动；重要按钮加 `:active` 缩放。
- 图片：加 `mode="aspectFit"` 或合理模式；必要时提供 `binderror` → 兜底。

### 文件与目录
- 页面内逻辑保持单文件职责；共用工具抽取到 `miniprogram/utils/`。
- 云函数一函数一目录，`index.js/package.json/config.json` 三件套齐全。
- 文档：设计规范（DESIGN.md）与开发文档（DEVELOPMENT.md）放根目录。

### 提交与分支
- 分支模型：`main`（稳定）/`dev`（集成）/`feature/*`（功能）。
- 提交信息：`feat: xxx`、`fix: yyy`、`docs: zzz`、`refactor:`、`chore:` 等。
- 合并前需自测并在真机/模拟器做核心路径回归。

## 数据演进与迁移指南
### 新增 `nextPaymentDate`（建议）
1. 添加页与编辑页保存/更新时，在客户端根据 `firstDate + cycle` 计算出最近一次未来扣费日，写入 `subscriptions.nextPaymentDate`。
2. 云函数查询 `nextPaymentDate` ∈ [今天, 今天+3] 且 `isRemind = true` 的记录，避免云端二次推算。
3. 为 `nextPaymentDate` 建立索引，提升筛选性能。
4. 兼容旧数据：云函数首次运行时对没有该字段的记录做兜底计算并写回（可选脚本）。

### 分类扩展
1. 在添加页/编辑页的分类映射中补充新类目名称。
2. 首页 `categories` 列表添加新项并保持筛选逻辑不变。
3. 统计页聚合无需变更，会自动按新分类汇总。

## 统计计算细则
- 月度折算：
  - 周：`price * 4.33`
  - 月：`price`
  - 季：`price / 3`
  - 年：`price / 12`
  - 自定义：若仅单次，暂按 `price` 计；如需要更精细，后续引入周期长度计算。
- 买断制：统计页按 12 个月平摊；首页总支出为总额计入（保持与当前口径一致）。
- 即将续订：订阅制且 30 天内（包含自定义周期以 `endDate` 为基准）；按 `daysRemaining` 升序。
- 过期判定：自定义周期 `endDate < today` 或明确 `isCancelled`，设为 `isExpired`，不计入总支出并沉底。

## 安全与隐私
- 不记录用户敏感数据至日志（openid 仅用于云函数发送，不落盘于客户端日志）。
- 禁止在仓库中提交密钥与非公开链接；模板 ID 仅作为字符串常量使用。
- 上传图片大小控制在 200KB 左右，防止包体和流量浪费。

## 性能与包体
- 单页首屏图片懒加载，避免一次渲染过多卡片图片。
- 减少不必要的 `setData` 次数（批量更新）。
- 控制自定义组件数量与层级，减少布局重排。
- 包体控制：不引入大型第三方库；统计页使用纯 CSS 动效。

---

## 可访问性与易用性
- 触控目标最小高度约 44px；按钮文本对比度符合黑白灰规范。
- 文本对齐左对齐；标题采用紧致行高，正文保持可读行高。
- 错误提示使用 Toast + Console（调试），关键错误可引导用户行动（如配置合法域名）。

## 术语表
- 订阅制（sub）：按周期收费（周/月/季/年/自定义）。
- 买断制（buyout）：一次性付款，首页总支出计总额，统计按年平摊。
- 即将续订：距离下次扣费 ≤30 天，紧急 ≤3 天。
- 过期（expired）：自定义周期 `endDate < today` 或明确取消。
- 授权拉起：保存或更新成功后使用 `wx.requestSubscribeMessage`。

## 事件流与状态机（关键路径）
### 新增订阅（成功路径）
1. 打开添加页 → 触发 `onLoad` 初始化日期与热门列表
2. 点击图标位 → 打开半屏弹窗 → `openIconModal`
3. 输入关键词 → `onSearchApp`（500ms 防抖）→ 本地命中 + 远程检索 → 渲染网格
4. 选择图标或上传 → 回填 `appName/iconUrl/category` → 关闭弹窗
5. 点击“保存订阅” → 校验 → `db.add` → `.then` 拉起 `wx.requestSubscribeMessage`
6. `complete` → Toast “保存成功” → 1.5s 后返回首页 → 首页 `onShow()` 刷新

### 删除订阅
1. 点击卡片 → `handleItemClick` → ActionSheet（编辑/删除）
2. 选择删除 → `showModal` 二次确认 → `db.doc(id).remove()` → 成功后刷新

### 统计页
1. 首页顶部“查看统计” → 跳转
2. `onShow` 拉取数据 → 过滤过期 → 分类聚合、折算与百分比 → 动画播放

## 错误处理与提示规范
- 网络失败：不弹窗，Console 记录；界面降级到本地热门与别名命中，显示“未找到相关应用”轻提示。
- 云数据库失败：Toast `保存失败/加载失败` 并打印 `err.errMsg`。
- 上传失败：Toast `上传失败`，保持弹窗打开。
- 订阅消息授权：
  - 不论允许/拒绝，均 Toast 成功并返回（降低打断感）。

## 风险清单与对策
- 外网接口不可用（地域/证书/限流）
  - 对策：本地热门 + 别名优先命中；离线图标映射落地（P1）。
- 云函数模板字段不匹配
  - 对策：上线前对齐后台模板；失败时记录日志不阻断循环。
- 字体/度量差异（模拟器与真机）
  - 对策：使用更稳的高度与行高；避免依赖字体的 baseline 精准对齐。
- 滚动容器高度问题
  - 对策：父容器固定高度 + 子级 `flex:1;height:0;`，禁止父级 `overflow:hidden`。
