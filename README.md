# 有数记 · 订阅账单管理小程序

> 一款简洁优雅的微信小程序，帮助你轻松管理和追踪所有订阅制服务的周期性支出。

## 功能特性

- **订阅管理** — 添加、编辑、删除你的订阅服务记录（Netflix、Spotify、iCloud 等）
- **智能分析** — 自动计算月度总支出、年度预估、日均消费，可视化分类占比
- **续费提醒** — 临近扣款日期自动高亮提示，支持微信服务通知订阅
- **分类筛选** — 按影音娱乐、实用工具、学习办公、游戏等分类快速过滤
- **云端同步** — 基于微信云开发，数据安全存储在云数据库中
- **多方式登录** — 微信一键登录、手机验证码登录、邮箱密码登录

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端框架 | 微信小程序原生（WXML + WXSS + JS） |
| 后端服务 | 微信云开发（云函数 + 云数据库） |
| 数据库 | 云开发 MongoDB（集合：`subscriptions`） |
| 定时任务 | 云函数 `sendReminder`（续费到期提醒推送） |

## 本地开发

1. 克隆仓库
   ```bash
   git clone git@github-youshuji:xixixier/youshuji.git
   ```
2. 使用 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 导入项目
3. 在开发者工具中开通云开发环境
4. 在云开发控制台创建 `subscriptions` 数据库集合
5. 部署 `cloudfunctions/sendReminder` 云函数

## 设计理念

界面采用温暖极简主义风格，以粉垩砂白（`#FAF9F6`）为基底，搭配沉稳的曜岩翠绿（`#0A5C43`）品牌色，追求优雅克制的视觉体验。拒绝多余的卡片阴影与高饱和色块，通过精确的间距节奏与纤细的排版层级传递高级感。
